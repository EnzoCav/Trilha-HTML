O que é HTML

(voltar ao indice)

HTML existe desde 1991 criado por Tim Berners-Lee considerado ''o pai da WEB''. Esse novo recurso criado por ele veio recheado de recursos e funcionalidades que trazem melhorias para o desenvolvimento web, seu recurso principal é demarcar a estrutura de uma página da web. A estrutura do HTML é formada por um conjunto de elementos, ou seja, os hipertextos, que se conectam entre si formando a página. Os elementos de HTML conhecidos como tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, etc.

Como Funciona o HTML

(voltar ao indice)

O HTML é uma linguagem de marcação linguagens essas que são constituídas de códigos que contem conteúdos específicos. Estes códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site. No início era muito complicado aprender HTML, pois eram muitos comandos para fazer algo simples. A cada nova versão, o HTML fica mais fácil de utilizar, hoje a mais usada é o HTML5. Atualmente qualquer pessoa pode acessar a internet e aprender a construir um site básico em questão de horas, seguindo os passos de tutoriais e aprendendo as funções de cada código. O HTML foi a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web, porém o HTML ainda prevalece. Atualmente já é possível integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site. Como por exemplo o HTML, CSS e JavaScrip Geralmente sempre estão lado a lado.

Tags de HTML

(voltar ao indice)

Os documentos HTML são arquivos de texto simples, com a extensão .html, e podem ser escritos na maioria dos editores de texto comuns, como o Bloco de Notas, do Windows. Em aplicações mais avançadas, recomenda-se a utilização de ambientes de desenvolvimento integrado, como o Visual Studio Code, que oferecem recursos que facilitam a escrita do código.

Alguns códigos em HTML e suas funções:

title - Define o titulo da pagina.

p - Define um paragrafo.

style - Define formatação em CSS.

h1 - h6 Define o nivel de Titulos do HTML.

blockquote - Define uma citação ao um texto ou marcação.

ol - Define uma lista Ordenada.

a - Define uma ancora e com o atributo href cria-se um hiperligação nas páginas web.
